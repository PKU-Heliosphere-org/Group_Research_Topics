球壳上在 Helmholtz 投影（Poisson 解）校正清楚磁场散度的数值方案

下面把在球壳体积上对 B(c)\mathbf B^{(c)}B(c) 做 Helmholtz 投影（解 Poisson 方程 ∇2Φ=∇ ⁣⋅B(c)\nabla^2\Phi=\nabla\!\cdot\mathbf B^{(c)}∇2Φ=∇⋅B(c) 并取 B=B(c)−∇Φ\mathbf B=\mathbf B^{(c)}-\nabla\PhiB=B(c)−∇Φ）的完整、可实现数值方案写清楚。重点采用对球面角用球谐分解（spherical harmonic）、径向用有限差分（或谱） 的混合谱-径向方法——这种方法在球壳上既准确又高效，且易于并行化。下面给出连续方程、每一步的离散化、边界条件选择、l=0 的特殊处理、实现要点与伪码/实现提示。

1. 问题与目标（回顾）

要解的方程（在球壳域 r∈[rin,rout], θ∈[0,π], ϕ∈[0,2π)r\in[r_{\rm in},r_{\rm out}],\ \theta\in[0,\pi],\ \phi\in[0,2\pi)r∈[rin​,rout​], θ∈[0,π], ϕ∈[0,2π)）：

∇2Φ(r,θ,ϕ)=ρ(r,θ,ϕ),ρ≡∇ ⁣⋅B(c).\nabla^2\Phi(r,\theta,\phi)=\rho(r,\theta,\phi),\qquad \rho\equiv\nabla\!\cdot\mathbf B^{(c)}.∇2Φ(r,θ,ϕ)=ρ(r,θ,ϕ),ρ≡∇⋅B(c).

投影后场

B=B(c)−∇Φ,∇ ⁣⋅B=0.\mathbf B=\mathbf B^{(c)}-\nabla\Phi,\quad\nabla\!\cdot\mathbf B=0.B=B(c)−∇Φ,∇⋅B=0.

我们选择对 (θ,ϕ)(\theta,\phi)(θ,ϕ) 做球谐变换，写

Φ(r,θ,ϕ)=∑ℓ=0Lmax⁡∑m=−ℓℓϕℓm(r) Yℓm(θ,ϕ),ρ(r,θ,ϕ)=∑ℓmρℓm(r) Yℓm(θ,ϕ).\Phi(r,\theta,\phi)=\sum_{\ell=0}^{L_{\max}}\sum_{m=-\ell}^{\ell}\phi_{\ell m}(r)\,Y_{\ell m}(\theta,\phi), \qquad \rho(r,\theta,\phi)=\sum_{\ell m}\rho_{\ell m}(r)\,Y_{\ell m}(\theta,\phi).Φ(r,θ,ϕ)=ℓ=0∑Lmax​​m=−ℓ∑ℓ​ϕℓm​(r)Yℓm​(θ,ϕ),ρ(r,θ,ϕ)=ℓm∑​ρℓm​(r)Yℓm​(θ,ϕ).

利用球谐的正交性，原 PDE 对每对 (ℓ,m)(\ell,m)(ℓ,m) 退化为一个一维径向常系数二阶 ODE：

 1r2ddr(r2dϕℓmdr)−ℓ(ℓ+1)r2 ϕℓm(r)=ρℓm(r). \boxed{\ \frac{1}{r^2}\frac{d}{dr}(r^2\frac{d\phi_{\ell m}}{dr}) -\frac{\ell(\ell+1)}{r^2}\,\phi_{\ell m}(r) =\rho_{\ell m}(r).\ } r21​drd​(r2drdϕℓm​​)−r2ℓ(ℓ+1)​ϕℓm​(r)=ρℓm​(r). ​

等价写法：

ddr(r2ϕℓm′)−ℓ(ℓ+1)ϕℓm=r2ρℓm(r).\frac{d}{dr}(r^2\phi_{\ell m}')-\ell(\ell+1)\phi_{\ell m}=r^2\rho_{\ell m}(r).drd​(r2ϕℓm′​)−ℓ(ℓ+1)ϕℓm​=r2ρℓm​(r).

所以：问题转为对每个 (ℓ,m)(\ell,m)(ℓ,m) 在径向解一个二阶常微分方程。

2. 球谐系数 ρℓm(r)\rho_{\ell m}(r)ρℓm​(r) 的计算（离散网格）

  • 在给定三维网格上（通常是 Nr×Nθ×NϕN_r\times N_\theta\times N_\phiNr​×Nθ​×Nϕ​，θ\thetaθ 用 Gauss–Legendre 节点、ϕ\phiϕ 用均匀网格最合适），在每个径向层 rir_iri​：

    ρℓm(ri)=∫02π ⁣∫0πρ(ri,θ,ϕ) Yℓm∗(θ,ϕ) sin⁡θ dθdϕ.\rho_{\ell m}(r_i)=\int_0^{2\pi}\!\int_0^\pi \rho(r_i,\theta,\phi)\,Y_{\ell m}^\ast(\theta,\phi)\,\sin\theta\,d\theta d\phi.ρℓm​(ri​)=∫02π​∫0π​ρ(ri​,θ,ϕ)Yℓm∗​(θ,ϕ)sinθdθdϕ.

  • 离散化用 Gauss-Legendre + trapezoidal in ϕ\phiϕ（或现成的 SHT 库，如 shtools, pyshtools，或基于 FFT + associated Legendre 机制），计算得到 ρℓm(ri)\rho_{\ell m}(r_i)ρℓm​(ri​) 对每个 iii 和 (ℓ,m)(\ell,m)(ℓ,m)。

实现要点：

  • 选 Lmax⁡L_{\max}Lmax​ 不要超过角向网格可解析的最大模式（奈奎斯特）：若 θ\thetaθ 使用 NθN_\thetaNθ​ 高斯点，通常 Lmax⁡≲Nθ−1L_{\max}\lesssim N_\theta-1Lmax​≲Nθ​−1；ϕ\phiϕ 网格 NϕN_\phiNϕ​ 决定能表示的 mmm 最大值。
  • 用库（例如 pyshtools、shtns）可大幅提升速度；若无库也可用 scipy.special.sph_harm 做直接投影（慢）。

3. 径向 ODE 的离散化与边界条件

3.1 连续 ODE：

ddr ⁣(r2ϕ′)−ℓ(ℓ+1)ϕ=r2ρℓm(r).\frac{d}{dr}\!\big(r^2\phi'\big)-\ell(\ell+1)\phi = r^2\rho_{\ell m}(r).drd​(r2ϕ′)−ℓ(ℓ+1)ϕ=r2ρℓm​(r).

3.2 边界条件的选择（实用建议）

常用两类 BC：

  • Neumann–Neumann（保边界法）：在内外边界保持法向场不变，即

    ∂Φ∂r∣r=rin=0,\frac{\partial\Phi}{\partial r}\bigg|_{r=r_{\rm{in}}}=0,∂r∂Φ​​r=rin​​=0,

    ∂Φ∂r∣r=rout=0,\frac{\partial\Phi}{\partial r}\bigg|_{r=r_{\rm{out}}}=0,∂r∂Φ​​r=rout​​=0,

    因为 Bn=Bn(c)−∂nΦB_n = B^{(c)}_n - \partial_n\PhiBn​=Bn(c)​−∂n​Φ，若要保持 BnB_nBn​ 在边界不变，需要 ∂nΦ=0\partial_n\Phi=0∂n​Φ=0.
    但 Neumann–Neumann 对应的 Poisson 问题需要兼容性条件：总体源积分为零（见下）。

  • Dirichlet–Neumann（常用且稳定）：例如在内边界设 Φ(rin)=0\Phi(r_{\rm in})=0Φ(rin​)=0（固定势，等价于允许边界法向 BBB 变化），外边界 Neumann（∂rΦ=0\partial_r\Phi=0∂r​Φ=0）或相反。Dirichlet–Neumann 消除了 Neumann 兼容性问题，但会改变边界法向分布（需要评估是否可接受）。

建议（若希望尽量不改动边界法向场）：优先尝试 Neumann–Neumann 且先检查兼容性（下文）。若兼容性差异很小（<10−12<10^{-12}<10−12），可以继续；若不满足，则采用 Dirichlet 在其中一侧或通过微小修正使兼容性成立（见 3.4）。

3.3 分析兼容性（Gauss 定理）

积分两边：

∫Vρ dV=∫V∇ ⁣⋅B(c) dV=∮∂VB(c) ⁣⋅ ⁣dS.\int_V \rho\,dV = \int_V \nabla\!\cdot\mathbf B^{(c)}\,dV = \oint_{\partial V}\mathbf B^{(c)}\!\cdot\!d\mathbf S .∫V​ρdV=∫V​∇⋅B(c)dV=∮∂V​B(c)⋅dS.

若选 Neumann–Neumann（∂nΦ=0\partial_n\Phi=0∂n​Φ=0 两边），则左边必须为 0 才有解。实际中若 B(c)\mathbf B^{(c)}B(c) 是通过 ∇×A\nabla\times A∇×A + 并行补偿得到，理论上应满足
∫V∇⋅B(c)dV=∮Bn(c)dS−∮Bn0dS\int_V \nabla\cdot\mathbf B^{(c)} dV = \oint B^{(c)}_n dS - \oint B^0_n dS∫V​∇⋅B(c)dV=∮Bn(c)​dS−∮Bn0​dS — 但数值截断、映射误差会产生小非零残差。

处理办法：

  • 直接计算 I=∫Vρ dVI=\int_V \rho\,dVI=∫V​ρdV. 若 ∣I∣|I|∣I∣ 小于容忍度（比如机器精度乘以体积或目标误差），直接继续 Neumann。
  • 若 III 不小，给 ρ\rhoρ 做微小调整：定义 ρ~=ρ−I/V\tilde\rho=\rho - I/ Vρ~​=ρ−I/V（均匀减后使体积分为 0）。这是最常见的数值修正（会把残差自由度平均分配到域内，改变极微小）。
  • 更理想的方法是修正边界法向：在边界上把 Bn(c)B^{(c)}_nBn(c)​ 做微小修正使通量匹配原始通量——这在工程上更复杂，一般不用，除非你必须严格保持边界 BnB_nBn​。

3.4 径向差分离散（二阶中心差分，均匀网格示例）

用径向网格 ri, i=0…Nr−1r_i,\ i=0\ldots N_r-1ri​, i=0…Nr​−1，间距 Δr\Delta rΔr（可用非均匀，但推导更长）。在点 rir_iri​ 用二阶差分离散方程：

定义 qi≡ri2q_i\equiv r_i^2qi​≡ri2​. 离散近似：

  • Neumann 在 i=0i=0i=0（内边界）写 ϕ′(r0)=0\phi'(r_0)=0ϕ′(r0​)=0 近似 (ϕ1−ϕ−1)/(2Δr)=0(\phi_1-\phi_{-1})/(2\Delta r)=0(ϕ1​−ϕ−1​)/(2Δr)=0；用虚点消除得到 ϕ−1=ϕ1\phi_{-1}=\phi_1ϕ−1​=ϕ1​。代入得到 i=0i=0i=0 的离散式（变系数）。

4. l=0 模式（球谐平均）的特殊处理

ℓ=0,m=0\ell=0,m=0ℓ=0,m=0 情况：
ODE 变为

1r2ddr ⁣(r2ϕ00′)=ρ00(r).\frac{1}{r^2}\frac{d}{dr}\!\big(r^2\phi_{00}'\big)=\rho_{00}(r).r21​drd​(r2ϕ00′​)=ρ00​(r).

积分得到

若选 Neumann BC 两端为 0，则需 ∫rinroutr2ρ00(r) dr=0\int_{r_{\rm in}}^{r_{\rm out}} r^2\rho_{00}(r)\,dr = 0∫rin​rout​​r2ρ00​(r)dr=0 才能兼容。实现上：

  • 若已用上面“均匀减法”保证总体体积分为 0，则该方程可解。
  • 否则可以对 ℓ=0\ell=0ℓ=0 用 Dirichlet（例如设 ϕ00(rin)=0\phi_{00}(r_{\rm in})=0ϕ00​(rin​)=0）再向外积分，这会允许调整边界通量。

5. 反变换并构造 ∇Φ\nabla\Phi∇Φ

  • 解出每个 ϕℓm(ri)\phi_{\ell m}(r_i)ϕℓm​(ri​) 后，在每个径向层重构 Φ(ri,θj,ϕk)\Phi(r_i,\theta_j,\phi_k)Φ(ri​,θj​,ϕk​) 用逆球谐变换：

    Φ(ri,θ,ϕ)=∑ℓmϕℓm(ri)Yℓm(θ,ϕ).\Phi(r_i,\theta,\phi)=\sum_{\ell m}\phi_{\ell m}(r_i) Y_{\ell m}(\theta,\phi).Φ(ri​,θ,ϕ)=ℓm∑​ϕℓm​(ri​)Yℓm​(θ,ϕ).

  • 计算梯度 ∇Φ\nabla\Phi∇Φ（球坐标形式）：

    ∂rΦ(ri,θj,ϕk)≈ϕ(ri+1)−ϕ(ri−1)2Δr,1r∂θΦ and 1rsin⁡θ∂ϕΦ\begin{aligned} \partial_r\Phi(r_i,\theta_j,\phi_k) &\approx \frac{\phi(r_{i+1})-\phi(r_{i-1})}{2\Delta r},\\ \frac{1}{r}\partial_\theta\Phi &\text{ and } \frac{1}{r\sin\theta}\partial_\phi\Phi \end{aligned}∂r​Φ(ri​,θj​,ϕk​)r1​∂θ​Φ​≈2Δrϕ(ri+1​)−ϕ(ri−1​)​, and rsinθ1​∂ϕ​Φ​

    其中 ∂θ,∂ϕ\partial_\theta,\partial_\phi∂θ​,∂ϕ​ 可由球谐系数解析求导（更精确且高效）：

    • 利用球谐的角导数表示：可直接在系数空间得到 ∂θYℓm\partial_\theta Y_{\ell m}∂θ​Yℓm​ 与 ∂ϕYℓm=imYℓm\partial_\phi Y_{\ell m}=im Y_{\ell m}∂ϕ​Yℓm​=imYℓm​ 的乘法关系，从而在系数层面直接算出角导数的重构系数，避免在实空间做差分。

  • 把 ∇Φ\nabla\Phi∇Φ 转为笛卡尔或直接以球坐标形式从 B(c)\mathbf B^{(c)}B(c) 中减去（两种都可）。

6. 算法总体流程（伪码） 

Given Bc on grid (r_i, theta_j, phi_k):

1. Compute divergence rho = div(Bc) on grid (use same finite-volume/discrete operator as Bc construction).

2. Check total integral I = ∑ rho * dV.
   if abs(I) > tol:
       # enforce compatibility (simple fix)
       rho = rho - I / V_total

3. For each radial layer r_i:
      compute spherical harmonic transform of rho(r_i, theta, phi) -> { rho_lm(r_i) } for all (l,m)

4. For each (l=0..Lmax, m=-l..l):
      assemble tridiagonal system for phi_lm(r) from discrete radial ODE
      apply BCs (Neumann/Dirichlet as chosen)
      solve for phi_lm(r) (Thomas or LAPACK)

5. For each radial layer r_i:
      reconstruct Phi(r_i,theta,phi) by inverse spherical harmonic transform from {phi_lm(r_i)}

6. Compute gradient ∇Phi on grid:
      - radial derivative by FD in r
      - angular derivatives via spherical harmonic coefficient relations or central differences
   Convert ∇Phi to Cartesian if needed.

7. Compute B = Bc - ∇Phi

8. Optionally: compute residual divergence of B for diagnostics (should be ~0 up to solver tolerance)

7. 实现细节、性能与数值建议

  • 球谐库：强烈建议用 shtns（C + Python 接口）或 pyshtools，因为直接用 scipy.special.sph_harm 在大网格上会很慢。若用 FFT+Legendre 配对法，角向变换复杂度约 O(NθNϕlog⁡Nϕ)O(N_\theta N_\phi \log N_\phi)O(Nθ​Nϕ​logNϕ​)。

  • 并行化：对不同 (ℓ,m)(\ell,m)(ℓ,m) 的径向方程可独立求解，或对不同 mmm 并行；角向 SHT 可用 MPI 并行分工。

  • 选择 Lmax⁡L_{\max}Lmax​：设 Lmax⁡≤min⁡(Nθ−1,  Nϕ/2)L_{\max}\leq \min(N_\theta-1,\;N_\phi/2)Lmax​≤min(Nθ​−1,Nϕ​/2)。实验上 Lmax⁡L_{\max}Lmax​ 取到 Nθ−1N_\theta-1Nθ​−1 会更精确但费时。

  • 径向网格：建议非均匀（如靠近内边界更细）以解析近日冕尺度；对应差分公式要用非均匀格点版本（上面给出均匀示例，但可扩展）。

  • 边界 BC 一致性：若你要把该场作为 MHD 初始场，一般希望保留内边界 BrB_rBr​（从观测给定），因此内边界应坚持 ∂rΦ=0\partial_r\Phi=0∂r​Φ=0（Neumann）或至少保证投影不改动已指定的法向值。外边界若是开边可选 Neumann。若 Neumann 兼容性难满足，用体积分布修正策略。

  • 验证：运行后检查两项关键量：

    1. max⁡∣∇ ⁣⋅B∣\max|\nabla\!\cdot\mathbf B|max∣∇⋅B∣ （应该接近 0, 例如 <10−10<10^{-10}<10−10 相对量），
    2. 点对点 ∣B∣−∣B0∣|\mathbf B|-|\mathbf B_0|∣B∣−∣B0​∣ 的误差（投影会引入小幅偏差，记录并观察是否可接受）。

  • 迭代精化：若投影后模长偏差太大，可做：并行补偿（用投影后 B 更新 b0hat）→再求解一次投影，迭代 1–3 次通常可以同时降低散度与保模误差。
